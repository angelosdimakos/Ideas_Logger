    scripts.refactor.parsers.docstring_parser¶

    Docstring Parser¶

    This module scans a Python project directory for missing or partial docstrings. It outputs structured JSON and markdown-style reports with description, args, and return sections. Also supports generating MkDocs-compatible markdown files.
